Category Command Description
Basic Commands mvn clean Removes target directory (previous build files)
mvn compile Compiles source code
mvn test Runs tests
mvn package Packages the compiled code into JAR/WAR
mvn install Builds and installs into local repository
mvn deploy Deploys project to remote repository
Lifecycle Phases mvn validate Validates project configuration
mvn compile Compiles source code
mvn test Runs tests
mvn package Creates JAR/WAR
mvn verify Verifies package meets quality standards
mvn install Installs package into local repository
mvn deploy Deploys to remote repository
Cleaning mvn clean Cleans up target directory
Dependency Management mvn dependency:resolve Resolves dependencies
mvn dependency:list Lists dependencies
mvn dependency:tree Shows dependency tree
mvn dependency:purge-local-repository Clears and re-downloads dependencies
Skipping Tests mvn install -DskipTests Skips tests during build
Running Specific Goals mvn <plugin>:<goal> Executes a specific plugin goal, e.g., mvn compiler:compile
Running Application (Spring Boot) mvn spring-boot:run Runs Spring Boot application (if applicable)
Maven Options mvn <command> -o Offline mode (no downloads)
mvn <command> -U Force update of dependencies
mvn <command> -X Debug mode for detailed logs
Project Information mvn help:effective-pom Displays effective POM after inheritance/interpolation
mvn help:active-profiles Shows all active profiles
