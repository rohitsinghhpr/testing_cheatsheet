Category Command Description
Configuration git config --global user.name "Your Name" Sets username
git config --global user.email "your.email@example.com" Sets email
git config --global --list Lists all global configs
Basic Commands git init Initializes a new repository
git clone <url> Clones a repository
git status Shows status of changes
git add <file> Stages file(s) for commit
git add . Stages all changes in current directory
git commit -m "message" Commits with message
git push Pushes changes to remote repository
git pull Fetches and merges from remote repository
git fetch Fetches changes without merging
Branching git branch Lists all branches
git branch <branch_name> Creates a new branch
git checkout <branch_name> Switches to a branch
git checkout -b <branch_name> Creates and switches to new branch
git merge <branch_name> Merges branch into current branch
git branch -d <branch_name> Deletes branch locally
Stashing git stash Stashes changes for later
git stash apply Applies the latest stash
git stash list Lists all stashes
git stash drop Removes latest stash
Viewing History git log Shows commit history
git log --oneline Shows concise history
git log -p Shows changes introduced by each commit
git show <commit_hash> Shows details of a specific commit
Undo Changes git reset <file> Unstages file
git reset --hard Resets working directory to last commit
git reset --soft HEAD~1 Undoes last commit, keeps changes staged
git revert <commit_hash> Reverts specific commit
Working with Remotes git remote -v Lists remote repositories
git remote add origin <url> Adds new remote repository
git push -u origin <branch_name> Pushes branch and sets upstream
git push origin --delete <branch_name> Deletes remote branch
Tagging git tag <tag_name> Creates a new tag
git tag Lists all tags
git push origin <tag_name> Pushes tag to remote repository
git tag -d <tag_name> Deletes local tag
git push origin --delete <tag_name> Deletes remote tag
Helpful Options git diff Shows unstaged changes
git diff --staged Shows staged changes
git diff <branch1> <branch2> Shows diff between two branches
git checkout -- <file> Discards changes to a file
