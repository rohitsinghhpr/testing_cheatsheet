Overview

The java.net.URL class represents a Uniform Resource Locator (URL), which is a pointer to a "resource" on the World Wide Web. A resource can be a file, a directory, or a reference to a more complicated object, such as a query to a database or a search engine.

Constructors

  • URL(String spec):

    Creates a URL object from a specified string representation.

  • URL(String protocol, String host, String file):

    Creates a URL with a specified protocol, host, and file path.

  • URL(String protocol, String host, int port, String file):

    Allows you to specify a port number along with the protocol, host, and file.

  • URL(URL context, String spec):

    Creates a URL by resolving a relative URL against a specified context.

Important Methods

  • getProtocol():

    Returns the protocol used by the URL (e.g., http, https, ftp).

  • getHost():

    Returns the host name of the URL.

  • getPort():

    Returns the port number associated with the URL. If no port is specified, it returns -1.

  • getFile():

    Returns the file name or path of the URL.

  • getPath():

    Returns the path part of the URL.

  • getQuery():

    Returns the query part of the URL (the part after the ?).

  • getRef():

    Returns the reference (anchor) part of the URL (the part after the #).

  • toURI():

    Converts the URL to a java.net.URI object.

Example Code

import java.net.URL;

public class URLExample {
    public static void main(String[] args) {
        try {
            URL url = new URL("https://www.example.com:8080/path/to/resource?query=java#section");
            System.out.println("Protocol: " + url.getProtocol());
            System.out.println("Host: " + url.getHost());
            System.out.println("Port: " + url.getPort());
            System.out.println("Path: " + url.getPath());
            System.out.println("Query: " + url.getQuery());
            System.out.println("Ref: " + url.getRef());
        } catch (Exception e) {
            e.printStackTrace();
        }
    }
}

Key Points

  • The URL class is used to create and manipulate URL objects.
  • It is part of the java.net package.
  • The methods allow you to parse and retrieve different parts of a URL.
