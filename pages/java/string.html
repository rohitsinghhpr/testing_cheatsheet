What is a String in Java?
  • Definition: A String in Java is a sequence of characters, represented as an object of the String class.
  • Immutable: Strings are immutable, meaning that once a String object is created, its value cannot be changed.
  • Storage: Strings are stored in a special memory area known as the "string pool."
What is the difference between String, StringBuilder, and StringBuffer?
  • String: Immutable and stored in the string pool.
  • StringBuilder: Mutable, not synchronized, and faster than StringBuffer.
  • StringBuffer: Mutable and synchronized, making it thread-safe but slightly slower than StringBuilder.
What is string immutability in Java and why is it important?
  • Immutability: Once created, a String object’s value cannot be changed.
  • Importance: Provides security, thread-safety, efficient memory usage, and helps in optimizing the string pool usage.
How does the Java string pool work?
  • String Pool: A special memory region in the Java heap that stores string literals.
  • Purpose: Reduces memory usage by storing only one instance of each literal.
  • Working: When a new string literal is created, Java checks the pool first; if it exists, the reference is returned, else a new string is added to the pool.
What is the difference between == and .equals() in Java for comparing strings?
  • == Operator: Compares references, i.e., checks if both references point to the same object in memory.
  • equals() Method: Compares the content of two strings to see if they have the same sequence of characters.
How are strings handled in memory when concatenation occurs?
  • Concatenation: When using + for concatenation, Java creates a new string because strings are immutable.
  • StringBuilder: For efficiency, the compiler often uses StringBuilder behind the scenes for repeated concatenation in loops.
  • Performance: Using StringBuilder or StringBuffer for concatenation is more memory-efficient.
Explain the intern() method in the String class.
  • intern() Method: Places a string in the string pool if it isn’t already there and returns the reference from the pool.
  • Usage: Can be used to save memory and increase performance by reusing strings from the pool.
What is the difference between length() and length in Java?
  • length(): Method used for finding the length of a String object.
  • length: Field used for finding the size of an array.
How do you convert a String to a primitive data type (e.g., int, double) in Java?
  • Using Wrapper Classes: Each primitive type has a wrapper class with parse methods (e.g., Integer.parseInt() for int, Double.parseDouble() for double).
  • Example: int num = Integer.parseInt("123");
