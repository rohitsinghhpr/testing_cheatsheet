1. What is an Array?

An array is a container object that holds a fixed number of values of a single type. In Java, arrays are objects that can store multiple elements of the same data type.

2. Declaration of Arrays

To declare an array in Java:


        int[] myArray;  // Recommended
        // or
        int myArray[];

3. Initialization of Arrays

Once declared, arrays can be initialized as follows:


        int[] myArray = new int[5]; // Array with 5 elements, default values are 0.
        myArray[0] = 10;           // Assign values to elements
        int[] anotherArray = {1, 2, 3, 4, 5}; // Inline initialization

4. 2D Arrays

A 2D array in Java is an array of arrays, useful for representing tables or matrices:


        int[][] matrix = new int[2][3]; // 2 rows, 3 columns
        matrix[0][0] = 1;
        matrix[0][1] = 2;
        matrix[0][2] = 3;
        int[][] anotherMatrix = {{1, 2}, {3, 4}, {5, 6}}; // Inline initialization

5. Annotations with Arrays

Annotations in Java can work with arrays. For example:


        @interface MyAnnotation {
            String[] value();
        }
        
        @MyAnnotation(value = {"Option1", "Option2"})
        public class MyClass {
        }

6. Methods and Arrays

Methods can accept arrays as parameters and return arrays:


        public static int[] reverseArray(int[] array) {
            int[] reversed = new int[array.length];
            for (int i = 0; i < array.length; i++) {
                reversed[i] = array[array.length - 1 - i];
            }
            return reversed;
        }

7. Reversing an Array

Example of reversing an array:


        int[] array = {1, 2, 3, 4, 5};
        int[] reversed = reverseArray(array);
        System.out.println(Arrays.toString(reversed)); // Output: [5, 4, 3, 2, 1]

8. Sorting an Array

Java provides utilities to sort arrays:


        import java.util.Arrays;
        
        int[] array = {5, 3, 2, 4, 1};
        Arrays.sort(array); // Sorts in ascending order
        System.out.println(Arrays.toString(array)); // Output: [1, 2, 3, 4, 5]

Conclusion

Arrays are powerful and versatile structures in Java, allowing efficient storage and manipulation of data. Mastery of arrays is essential for working with Java effectively.
