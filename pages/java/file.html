1. File Class

The File class in Java is used to represent files and directories. It provides methods to create, delete, and obtain file information.

import java.io.File;

public class FileExample {
    public static void main(String[] args) {
        // Create a File object for a specific file path
        File file = new File("example.txt");

        // Check if the file exists
        if (file.exists()) {
            System.out.println("File exists.");
        } else {
            System.out.println("File does not exist.");
        }

        // Get file information
        System.out.println("File name: " + file.getName());        // Get file name
        System.out.println("Path: " + file.getPath());             // Get file path
        System.out.println("Absolute path: " + file.getAbsolutePath());  // Get absolute path
        System.out.println("Writable: " + file.canWrite());        // Check if writable
        System.out.println("Readable: " + file.canRead());         // Check if readable
        System.out.println("File size in bytes: " + file.length()); // Get file size

        // Create a new file
        try {
            boolean isCreated = file.createNewFile();              // Create the file
            if (isCreated) {
                System.out.println("File created.");
            }
        } catch (Exception e) {
            System.out.println("An error occurred while creating the file.");
        }

        // Delete the file
        boolean isDeleted = file.delete();
        if (isDeleted) {
            System.out.println("File deleted.");
        }
    }
}

2. FileWriter Class

The FileWriter class is used to write character-based data to a file. It allows writing strings or characters and is often used with a BufferedWriter for efficiency.

import java.io.FileWriter;
import java.io.IOException;

public class FileWriterExample {
    public static void main(String[] args) {
        try {
            FileWriter writer = new FileWriter("output.txt"); // Create FileWriter object to write to "output.txt"
            writer.write("Hello, World!\n");                  // Write a string
            writer.write("Welcome to FileWriter in Java.");   // Write another string
            writer.close();                                   // Close the writer to free resources
            System.out.println("Successfully written to file.");
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}

3. FileReader Class

The FileReader class is used to read character-based data from a file. This is ideal for text files and allows reading character by character or in bulk.

import java.io.FileReader;
import java.io.IOException;

public class FileReaderExample {
    public static void main(String[] args) {
        try {
            FileReader reader = new FileReader("output.txt");  // Create FileReader to read "output.txt"
            int character;

            // Read characters until the end of the file
            while ((character = reader.read()) != -1) {
                System.out.print((char) character);           // Convert integer to character and print
            }
            reader.close();                                   // Close the reader
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}

4. FileInputStream Class

The FileInputStream class is used to read binary data from a file, making it suitable for reading files like images, audio, and video. It reads data byte by byte.

import java.io.FileInputStream;
import java.io.IOException;

public class FileInputStreamExample {
    public static void main(String[] args) {
        try {
            FileInputStream fis = new FileInputStream("example.bin");  // Create FileInputStream for binary file
            int byteData;

            // Read bytes until the end of the file
            while ((byteData = fis.read()) != -1) {
                System.out.print(byteData + " ");                      // Print each byte as an integer
            }
            fis.close();                                               // Close the stream
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}

5. FileOutputStream Class

The FileOutputStream class is used to write binary data to a file. It writes data byte by byte and is commonly used for files like images, audio, and videos.

import java.io.FileOutputStream;
import java.io.IOException;

public class FileOutputStreamExample {
    public static void main(String[] args) {
        try {
            FileOutputStream fos = new FileOutputStream("output.bin"); // Create FileOutputStream to write binary data
            fos.write(65);   // Write the byte (65 corresponds to 'A')
            fos.write(66);   // Write the byte (66 corresponds to 'B')
            fos.write(67);   // Write the byte (67 corresponds to 'C')
            fos.close();     // Close the stream to save resources
            System.out.println("Binary data written to file.");
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}

Summary of Key Methods

  1. File:

    • exists(): Checks if the file exists.
    • createNewFile(): Creates a new file.
    • delete(): Deletes the file.
    • canRead(), canWrite(): Checks read/write permissions.
    • getName(), getPath(), getAbsolutePath(): Retrieves file information.
    • length(): Gets the file size in bytes.

  2. FileWriter:

    • write(String s): Writes a string to a file.
    • close(): Closes the writer.

  3. FileReader:

    • read(): Reads a single character as an integer.
    • close(): Closes the reader.

  4. FileInputStream:

    • read(): Reads a single byte as an integer.
    • close(): Closes the input stream.

  5. FileOutputStream:

    • write(int b): Writes a single byte to the file.
    • close(): Closes the output stream.

Each class in Java's file handling API provides a way to work with files and data for specific needs (text vs. binary). These examples cover the basic usage of each, and they can be extended or combined depending on application requirements.
