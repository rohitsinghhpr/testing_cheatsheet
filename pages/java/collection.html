ArrayList

Overview

An ArrayList is a resizable array, part of the java.util package. It is best suited for dynamic arrays where frequent additions or removals occur.

Declaration and Initialization


        import java.util.ArrayList;
        
        ArrayList list = new ArrayList<>(); // Default capacity
        ArrayList numbers = new ArrayList<>(10); // Initial capacity

Common Methods

  • add(element) - Adds an element to the list.
  • get(index) - Retrieves the element at the specified index.
  • remove(index) - Removes the element at the specified index.
  • size() - Returns the number of elements in the list.

Example Usage


        import java.util.ArrayList;
        
        ArrayList fruits = new ArrayList<>();
        fruits.add("Apple");
        fruits.add("Banana");
        fruits.add("Cherry");
        
        System.out.println(fruits.get(1)); // Output: Banana
        fruits.remove(0);
        System.out.println(fruits.size()); // Output: 2
LinkedList

Overview

A LinkedList is a doubly-linked list implementation. It is optimal for scenarios involving frequent insertions and deletions.

Declaration and Initialization


        import java.util.LinkedList;
        
        LinkedList list = new LinkedList<>();

Common Methods

  • addFirst(element) - Adds an element at the beginning of the list.
  • addLast(element) - Adds an element at the end of the list.
  • removeFirst() - Removes the first element.
  • getLast() - Retrieves the last element.

Example Usage


        import java.util.LinkedList;
        
        LinkedList colors = new LinkedList<>();
        colors.add("Red");
        colors.addFirst("Blue");
        colors.addLast("Green");
        
        System.out.println(colors.getLast()); // Output: Green
        colors.removeFirst();
        System.out.println(colors); // Output: [Red, Green]
HashSet

Overview

A HashSet is a collection that does not allow duplicate elements. It is implemented using a hash table and offers constant time performance for basic operations.

Declaration and Initialization


        import java.util.HashSet;
        
        HashSet set = new HashSet<>();

Common Methods

  • add(element) - Adds an element to the set.
  • contains(element) - Checks if the set contains the specified element.
  • remove(element) - Removes the specified element from the set.
  • size() - Returns the number of elements in the set.

Example Usage


        import java.util.HashSet;
        
        HashSet animals = new HashSet<>();
        animals.add("Dog");
        animals.add("Cat");
        animals.add("Dog"); // Duplicate will be ignored
        
        System.out.println(animals.contains("Cat")); // Output: true
        animals.remove("Dog");
        System.out.println(animals.size()); // Output: 1
HashMap

Overview

A HashMap is a key-value pair collection. It does not allow duplicate keys but allows multiple null values.

Declaration and Initialization


        import java.util.HashMap;
        
        HashMap map = new HashMap<>();

Common Methods

  • put(key, value) - Adds a key-value pair to the map.
  • get(key) - Retrieves the value associated with the specified key.
  • remove(key) - Removes the key-value pair associated with the specified key.
  • containsKey(key) - Checks if the map contains the specified key.

Example Usage


        import java.util.HashMap;
        
        HashMap scores = new HashMap<>();
        scores.put("Alice", 90);
        scores.put("Bob", 85);
        
        System.out.println(scores.get("Alice")); // Output: 90
        scores.remove("Bob");
        System.out.println(scores.containsKey("Bob")); // Output: false
