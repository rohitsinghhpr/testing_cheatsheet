GET Request
A GET request is used to retrieve data from the server at a specified resource endpoint: 

            import static io.restassured.RestAssured.*;
            import io.restassured.response.Response;
            
            public class ApiTest {
                public static void main(String[] args) {
                    Response response = given()
                        .when()
                        .get("https://jsonplaceholder.typicode.com/posts/1");
            
                    System.out.println("Status Code: " + response.getStatusCode());
                    System.out.println("Response Body: " + response.getBody().asString());
                }
            }
This example sends a GET request and prints the status code and response body.
POST Request
A POST request is used to send data to the server to create a new resource: 

            import static io.restassured.RestAssured.*;
            import io.restassured.http.ContentType;
            
            public class ApiTest {
                public static void main(String[] args) {
                    given()
                        .contentType(ContentType.JSON)
                        .body("{ \"title\": \"foo\", \"body\": \"bar\", \"userId\": 1 }")
                        .when()
                        .post("https://jsonplaceholder.typicode.com/posts")
                        .then()
                        .statusCode(201);
                }
            }
This sends a POST request with JSON data to create a new resource and verifies that the response status code is 201.
PUT Request
A PUT request is used to update an existing resource with the data provided in the request: 

            import static io.restassured.RestAssured.*;
            import io.restassured.http.ContentType;
            
            public class ApiTest {
                public static void main(String[] args) {
                    given()
                        .contentType(ContentType.JSON)
                        .body("{ \"title\": \"foo updated\", \"body\": \"bar updated\", \"userId\": 1 }")
                        .when()
                        .put("https://jsonplaceholder.typicode.com/posts/1")
                        .then()
                        .statusCode(200);
                }
            }
This sends a PUT request to update an existing resource and checks that the response status code is 200.
PATCH Request
A PATCH request is used to partially update an existing resource with only the provided data: 

            import static io.restassured.RestAssured.*;
            import io.restassured.http.ContentType;
            
            public class ApiTest {
                public static void main(String[] args) {
                    given()
                        .contentType(ContentType.JSON)
                        .body("{ \"title\": \"foo patched\" }")
                        .when()
                        .patch("https://jsonplaceholder.typicode.com/posts/1")
                        .then()
                        .statusCode(200);
                }
            }
This example sends a PATCH request to update only the title field of an existing resource and validates the response code.
DELETE Request
A DELETE request is used to remove a specified resource from the server: 

            import static io.restassured.RestAssured.*;
            
            public class ApiTest {
                public static void main(String[] args) {
                    given()
                        .when()
                        .delete("https://jsonplaceholder.typicode.com/posts/1")
                        .then()
                        .statusCode(200); // or 204 based on API behavior
                }
            }
This sends a DELETE request to remove a resource and checks for a 200 (OK) or 204 (No Content) status code.
HEAD Request
A HEAD request retrieves the headers from a resource without the response body, often used to check metadata: 

            import static io.restassured.RestAssured.*;
            
            public class ApiTest {
                public static void main(String[] args) {
                    given()
                        .when()
                        .head("https://jsonplaceholder.typicode.com/posts/1")
                        .then()
                        .statusCode(200)
                        .header("Content-Type", "application/json; charset=utf-8");
                }
            }
This example sends a HEAD request to verify headers and checks that the response code and Content-Type header are as expected.
OPTIONS Request
An OPTIONS request retrieves the HTTP methods supported by a specific URL: 

            import static io.restassured.RestAssured.*;
            
            public class ApiTest {
                public static void main(String[] args) {
                    given()
                        .when()
                        .options("https://jsonplaceholder.typicode.com/posts/1")
                        .then()
                        .statusCode(200)
                        .header("Allow", containsString("GET"));
                }
            }
This sends an OPTIONS request and checks if the Allow header contains the allowed HTTP methods, such as GET.
