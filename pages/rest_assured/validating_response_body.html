Validating a Specific Field in the Response Body
Checking if a particular field in the JSON response body matches an expected value: 

            import static io.restassured.RestAssured.*;
            import static org.hamcrest.Matchers.*;
            
            public class ApiTest {
                public static void main(String[] args) {
                    given()
                        .when()
                        .get("https://jsonplaceholder.typicode.com/posts/1")
                        .then()
                        .body("userId", equalTo(1))
                        .body("id", equalTo(1))
                        .body("title", equalTo("sunt aut facere repellat provident occaecati excepturi optio reprehenderit"));
                }
            }
This example validates that userId, id, and title fields in the JSON response match specific values.
Validating Multiple Fields with Logical Conditions
Using logical conditions to verify multiple fields in the response body: 

            import static io.restassured.RestAssured.*;
            import static org.hamcrest.Matchers.*;
            
            public class ApiTest {
                public static void main(String[] args) {
                    given()
                        .when()
                        .get("https://jsonplaceholder.typicode.com/posts/1")
                        .then()
                        .body("userId", greaterThan(0))
                        .body("id", lessThanOrEqualTo(100));
                }
            }
This checks that userId is greater than 0 and id is less than or equal to 100.
Validating JSON Array Elements
Validating specific elements in a JSON array within the response body: 

            import static io.restassured.RestAssured.*;
            import static org.hamcrest.Matchers.*;
            
            public class ApiTest {
                public static void main(String[] args) {
                    given()
                        .when()
                        .get("https://jsonplaceholder.typicode.com/posts")
                        .then()
                        .body("userId", hasItems(1, 2, 3));
                }
            }
This example checks if the userId array in the response body contains the values 1, 2, and 3.
Validating Nested JSON Fields
Validating nested fields in a JSON response: 

            import static io.restassured.RestAssured.*;
            import static org.hamcrest.Matchers.*;
            
            public class ApiTest {
                public static void main(String[] args) {
                    given()
                        .when()
                        .get("https://jsonplaceholder.typicode.com/comments/1")
                        .then()
                        .body("postId", equalTo(1))
                        .body("name", containsString("id labore ex et quam laborum"));
                }
            }
This example verifies the postId and a specific name field in a nested JSON response.
Validating JSON Body with Regular Expressions
Using regular expressions to validate fields in the response body: 

            import static io.restassured.RestAssured.*;
            import static org.hamcrest.Matchers.*;
            
            public class ApiTest {
                public static void main(String[] args) {
                    given()
                        .when()
                        .get("https://jsonplaceholder.typicode.com/comments/1")
                        .then()
                        .body("email", matchesPattern("^[A-Za-z0-9._%+-]+@[A-Za-z0-9.-]+\\.[A-Za-z]{2,6}$"));
                }
            }
This validates that the email field in the response matches a valid email format using a regular expression.
Validating Full Response Body with JSONPath
Verifying the entire structure of a response using JSONPath: 

            import static io.restassured.RestAssured.*;
            import static org.hamcrest.Matchers.*;
            
            public class ApiTest {
                public static void main(String[] args) {
                    given()
                        .when()
                        .get("https://jsonplaceholder.typicode.com/posts/1")
                        .then()
                        .body("$", hasKey("userId"))
                        .body("$", hasKey("id"))
                        .body("$", hasKey("title"))
                        .body("$", hasKey("body"));
                }
            }
This example checks that the JSON response contains the keys userId, id, title, and body.
